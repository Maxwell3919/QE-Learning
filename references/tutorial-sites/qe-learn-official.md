# QE Learn Official

## 1. 它是什么？

QE 官方 Learn / hands-on materials 是官方生态中的教学入口，适合作为官方背景材料来源。

## 2. 适合学习或解决什么问题？

适合确认官方推荐的教程路线、学校材料和 hands-on 入口。

## 3. 它覆盖哪些 QE workflow？

覆盖范围随官方页面更新，通常围绕 SCF、PWscf、PHonon、post-processing 和学校材料。

## 4. 哪些部分适合跟做？

官方 hands-on 例子适合跟做，但应同步写入本仓库 `cases/` 和 `record.md`。

## 5. 哪些部分只适合作为参考？

官方页面的总览信息适合索引，不等于完整学习路线。

## 6. 它和本仓库哪些页面对应？

- `learn/00-start-here.md`
- `references/source-index.md`

## 7. 它的局限是什么？

官方入口通常按资源列举，不一定显式说明 workflow DAG 和 PASS/WARN/BLOCK 判断。
