# PLUMED

## 1. 它是什么？

PLUMED 是增强采样和 metadynamics 工具，可与 QE 生态中的动力学分支关联。

## 2. 适合学习或解决什么问题？

适合高级分子动力学、自由能面和增强采样。

## 3. 它覆盖哪些 QE workflow？

molecular dynamics / metadynamics 分支。
