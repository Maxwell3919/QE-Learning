# AiiDAlab QE

## 1. 它是什么？

AiiDAlab QE app 是基于 AiiDA 的交互式 QE workflow 应用。

## 2. 适合学习或解决什么问题？

适合理解 GUI 如何包装 provenance-aware QE workflow。

## 3. 它覆盖哪些 QE workflow？

结构输入、workflow 选择、计算运行、结果面板和插件式扩展。

## 4. 哪些部分适合跟做？

适合在掌握手动 QE 和 AiiDA 基础后体验。

## 5. 哪些部分只适合作为参考？

GUI 交互不能替代对 input/output 和 convergence 的理解。

## 6. 它和本仓库哪些页面对应？

- `learn/07-hpc-record-provenance-loop.md`
- `references/tools/aiida-qe.md`

## 7. 它的局限是什么？

它提升交互体验，但会隐藏部分文件级细节，初学者仍需回到 QE input/output。
