# Electronic Structure Loop

## 能力目标

掌握 QE 电子结构主线：bands、DOS 和 PDOS。

## 完成判据

- 能完成 `SCF -> bands -> bands.x`。
- 能完成 `SCF -> NSCF -> dos.x`。
- 能完成 `projwfc.x` 的 PDOS 基本流程。
- 能说明 k-path 与 uniform k mesh 的区别。
- 能说明 Fermi level、VBM、CBM、band gap 的基本判断。

## 可验证输出

- bands k-path 来源说明。
- DOS uniform k mesh 设置说明。
- energy zero / Fermi level 说明。
- bands/DOS/PDOS output 阅读笔记。

## 推荐阅读

- QE INPUT_BANDS reference
- QE DOS / PROJWFC input references
- Pranab Das bands / DOS / PDOS materials
- SeeK-path documentation

## 后续进入哪个 workflow

完成后进入 postprocessing、Wannier90 overview 或高级电子结构分支。
